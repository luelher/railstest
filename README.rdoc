== README

Ejercicio práctico del uso de AngularJS con Rails 4.

Aplicación que genera:
- Series de fibonacci
- Primo Palindromos
- Número de "unos" (1) en un numero en formato binario.

== Instalación

* clone
* Instalar rvm 
* Entrar en la carpeta del proyecto
* instalar ruby
* instalar bundle
* bundle install
* rails s
